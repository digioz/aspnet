Install Instructions:

1- Place guestbook in a directory inside your wwwroot folder. 
2- Start Internet Information Services (IIS) Manager by going to Start > Control Panel > Administrative Tools > Internet Information Manager
3- Right-Click on the guestbook folder, choose properties. 
4- Hit the Create button to create an application name, and hit OK. 
5- Browse to the application folder using File Explorer, and select the App_Data folder by Right Clicking on the folder and choosing properties. 
6- Give the following two users Full Control Permission under the Security Tab: IIS_WPG (MACHINENAME\IIS_WPG) and Internet Guest Account (MACHINENAME\IUSR_MACHINENAME).

Enjoy!
